# Introdução ao React Native!

## Setup 

Para rodar esse projeto, será necessário instalar o [nvm](https://github.com/nvm-sh/nvm) e a versão lts do [Node](https://nodejs.org/pt-br).

No Linux, instalamos o nvm com o seguinte comando:
```
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
```
Agora, instalamos o Node:
```
nvm install --lts
```

Também vamos utilizar o gerenciador de pacotes [Yarn](https://yarnpkg.com/), que já vem instalado com o Node e deve ser habilitado:
```
corepack enable
```

## Rodando o projeto
Uma vez que o ambiente está configurado, clone o repositório:
```
git clone https://github.com/ederzin13/componentes-basicos
```
Navegue para a pasta:
```
cd componentes-basicos
```
E instale as dependências:
```
yarn
```

Agora, basta rodar **yarn start** e ler o QR Code pelo aplicativo Expo Go em seu celular.
